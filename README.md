Amazon Sales Analytics Dashboard

This project presents an interactive Amazon Sales Dashboard developed in Microsoft Excel with data cleaning and preprocessing performed using Python.

Project Overview

The dashboard provides insights into sales performance, customer activity, product trends, geographic distribution, and order behavior through multiple interactive reports.
